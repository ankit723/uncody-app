Editable Text

How To Use

  • Select the Text You Want to Edit
  • After Selecting the text, a toolbar will Show up with few options, followed as;
  • 1> Bold the selected Text
  • 2> Italic the selected Text
  • 3> Underline the selected Text
  • 4> Edit the selected Text
  • 5> TO change the size first select the size then select the text and click Ss
  • 6> To change the color first select the color you want then select the text and click Cc

Note! Following Editing Can be done any where in the Page
